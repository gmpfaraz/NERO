🔍 SEARCH COMMANDS CHEAT SHEET:

   starts:N    → Numbers starting with N
   ends:N      → Numbers ending with N
   middle:N    → N in the middle
   contains:N  → Contains N anywhere
   equals:N    → Exact match
   between:N-M → Numbers in range N to M
   even:       → Even numbers only
   odd:        → Odd numbers only
   sum:N       → Digit sum equals N
   length:N    → Numbers with length N
   greater:N   → Numbers > N
   less:N      → Numbers < N

   Wildcards still work:
   *           → All numbers
   1*          → Starts with 1
   *5          → Ends with 5
   *3*         → Contains 3
